# Crédits

## Sprites des personnages (pixel art)

Les feuilles de sprites d'Eneko (dossier `Samurai`) et de l'ennemi Yurei
proviennent du dépôt public **shinobi-game** :

- Source : https://github.com/alejandro-cartagena/shinobi-game
  (dossiers `SpriteSheets/Samurai`, `Kitsune`, `Shinobi` et `Onre`)
- Fichiers utilisés :
  - `assets/character/samurai/` — Eneko (Idle, Run, Walk, Jump, Attack_1, Hurt, Dead)
  - `assets/character/kitsune/` — Léonie (Idle, Walk)
  - `assets/enemies/shinobi/` — Ombre corrompue (Idle, Run, Attack_1, Dead)
  - `assets/enemies/onre/` — Esprit Onre (Idle, Walk, Attack_1, Dead)

Ces sprites sont utilisés ici dans le cadre d'un **projet personnel non
commercial**. Merci à l'auteur d'origine du pack pixel art. Si tu comptes
distribuer ou vendre le jeu, vérifie/complète la licence exacte du pack
auprès de son auteur avant publication.

## Reste du jeu

Décor (bambous, torii, lanternes, cœurs, HUD), placeholders et code
GDScript : réalisés pour ce projet.
