📘 README – Pixel DarkNinja

First, thank you for purchasing this pack!
This pack contains all the animations for a pixel DarkNinja character (32px).
Each animation is hand-drawn frame by frame, and organized by folders for easy import.

⚔️ CONTENTS

Idle – 8 frames 
Move – 8 frames 
Jump - 12 frames
Attack1 – 8 frames
Attack2 – 8 frames
Attack3 – 8 frames
Hurt – 3 frames
Defend - 8 frames

Each frame is placed inside a 64×64 grid, recommended 12 FPS playback speed.
Works well with Aseprite, Unity, Godot, GameMaker, etc.
